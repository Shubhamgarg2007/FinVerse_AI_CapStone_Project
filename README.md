## Setup the frontend

```sh
cd frontend
npm install
npm run dev
```
## Setup the backend

```sh
# (Optional) Create and activate a virtual environment
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the backend serve
uvicorn main:app --reload       
```
